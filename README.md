# Terraform Learnings (GCP)

This repository is dedicated to learning Terraform with a focus on Google Cloud Platform. It contains notes, experiments, and infrastructure code.

## Folder Structure

- `notes/`: Thought leadership, technology notes, and insights.
- `notebooks/`: Jupyter notebooks for interactive experimentation.
- `terraform/`: Terraform configurations.
  - `modules/`: Reusable GCP resource modules.
  - `environments/`: Environment-specific deployments (dev, prod, etc.).
- `scripts/`: Automation and helper scripts.
- `references/`: Storage for PDF books, whitepapers, and reference materials.
- `cloned-repos/`: External repositories cloned for study and reference.

## Getting Started

1. Set up your GCP credentials.
2. Explore the `notes/` for theoretical understanding.
3. Check `terraform/` for infrastructure examples.
